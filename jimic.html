Jimic

Jimic is a minimalistic eComic viewer for Java platforms supporting JDK 1.1 and AWT. It allows to read eComics stored in CBZ format with images encoded in JFIF/JPEG or GIF format. To read comics or images in other formats, they first have to be converted (for example using Jomic).

Features and limitations

Jimic is designed to be simple and small:

  • Maximum screen real estate due lack of menus, toolbars and anything else not related to showing images.
  • No write access needed. Images are read directly from the eComic ZIP archive, and extracted to memory.
  • Tiny application size.
  • Runs with JDK 1.1 and AWT (as released way back in 1997).

Because of its simplicity, a few things you might have been used to from other eComic viewers are missing. The limitations are by design, and not considered bugs.

  • No scrollbar to move within the image (because it would waste screen space).
  • No online help; you have to learn the keyboard definition by heart (it's simple though because there isn't much you can do with Jimic anyway).
  • No two page mode.
  • No caching and precaching.
  • Read only one comic at a time.
  • No different view modes like "scale to fit" (because handheld displays are usually to small for that anyway).

And there are a couple of issues that ought to be addressed eventually:

  • Scrolling should only be allowed within the image range.
  • Image names should be sorted naturally instead of alphabetically (23 comes before 123).

Requirements

Jimic requires a Java and its Abstract Windowing Toolkit (AWT). These have been available since 1997, and are supported by pretty much everything that claims Java with GUI support.

Installation and startup

To install Jimic, unpack Jimic.zip and simply copy the Jimic.jar to any place you like.

To start Jimic from the console, type:

java -jar /install/path/Jimic.jar

Optionally you can specify a file name of a comic to open, for example:

java -jar /install/path/Jimic.jar /comics/blah.cbz

On many platforms you can also start Jimic from the desktop by just double-clicking the Jimic.jar.

Usage

Jimic opens an empty window. Press one of the following keys:

  • O - Open a dialog to select a comic to read.
  • A - Show dialog with information about the application such as version number and copyright.
  • Page up, page down, Space - Go to previous/next image.
  • Cursor keys - scroll image.
  • G or L - Open a dialog to select the numer of an image to show.
  • Q or Esc - Quit the application.

Additionally you can use the mouse:

  • Click left - Go to next image.
  • Drag with left button pressed - Scroll image.

Source code

The source code in included in the Jimic.zip in the directory source. There also is a build.xml for ant. To compile the source code, run:

ant
To get information about other targets, run: 
ant -projecthelp
